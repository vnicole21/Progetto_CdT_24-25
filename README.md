# Progetto_CdT_24-25
Repository del progetto di Codifica di Testi, a.a. 24/25, per il corso di laurea di Informatica Umanistica dell'Università di Pisa.

L'output html è stato generato con questa riga di comando da terminale: java -jar SaxonHe12-9J\saxon-he-12.9.jar -s:tei.xml -xsl:codifica.xsl -o:output.html 
Questo ha permesso di verificare la correttezza del documento XML tramite Xerces e DTD: java -cp ".;xerces-2_12_2-xml-schema-1.1/*" dom.Counter -v tei.xml"
